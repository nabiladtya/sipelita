Nama Proyek: DigisionDev Website
DigisionDev Website .Build with boostrap, css, html, JavaScript, jquery, etc.
Deskripsi: 
Proyek ini adalah sebuah website yang dirancang untuk perusahaan DigisionDev.
Website ini menggunakan Bootstrap untuk desain responsif, PHP untuk menangani form dan menghubungkan ke database MySQL,
serta memiliki beberapa bagian seperti carousel, about, portofolio, dan kontak.

Fitur Pengguna:
1. Navbar
2. Landing Page Profil DigisionDev
3. Penawaran Produk
4. Tabel Harga Paket 
5. Komentar 
6. Link Vidio Promosi Social Media Youtube 
7. About Us / Tentang Kami
8. Visi dan Misi
9. galeri
10. Footer

Fitur Admin:
1. Login Page
2. kode Captcha
3. Salam dan Nama admin
4. Jam realtime
5. Create New Admin Account
6. Edit Account
7. Logout
8. Lihat Pesanan (CRUD)
9. Grafik pesanan
10. Lihat Komentar (CRUD)
